# tm_scripts_by_leerov

Коллекция пользовательских скриптов для Tampermonkey.

## Скрипты

### Убираем фон и скрываем скроллы
Убирает фон у `body`, `main`, `.root` и скрывает все полосы прокрутки на всех страницах.

## Установка

### Способ 1: Импорт всех скриптов (рекомендуется)
1. Откройте панель управления Tampermonkey.
2. Перейдите на вкладку «Утилиты».
3. В разделе «Импорт» вставьте URL:
   ```
   https://raw.githubusercontent.com/leerov/tm_scripts_by_leerov/main/out.json
   ```
4. Нажмите «Импорт».

### Способ 2: Установка отдельного скрипта
Перейдите по ссылке на файл скрипта (например, `script.js`) и установите вручную.

## Добавление нового скрипта

1. Создайте папку в `scripts/` с названием вашего скрипта.
2. Положите туда `script.js` (код без мета-блока) и `script.json` (метаданные).
3. Сделайте push — GitHub Actions автоматически соберёт новый `out.json`.

## Автоматическая сборка

При каждом пуше в `main` запускается GitHub Action, который собирает все скрипты в единый `out.json` для импорта.
